## TasteStack

TasteStack is a frontend food ordering web app currently being built using HTML, CSS, and JavaScript.

This project focuses on learning real-world DOM manipulation, UI structuring, and frontend development practices.

---

## Learning Goals

- Practice JavaScript fundamentals
- Understand DOM manipulation
- Build dynamic UI using JS
- Learn cart logic step-by-step
- Follow real developer workflow

---

## Tech Stack 

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## Planned Features

- Dynamic food item rendering
- Add to cart functionality
- Remove items from cart
- Quantity management
- Total price calculation
- Search and filtering
- LocalStorage support
